# 🏆 منصة يوسف للتداول الاحترافية 📊

> تطبيق تداول متكامل مع إدارة مهام - منصة احترافية لتداول العملات الرقمية والذهب والعملات الأجنبية

## ✨ المميزات الرئيسية

### 📊 لوحة التحكم
- **إحصائيات فورية**: الرصيد الكلي، الأرباح اليومية، عدد العمليات
- **رسم بياني للأسعار**: تصور مرئي لتطور الأسعار
- **جدول آخر العمليات**: متابعة فورية لجميع عملياتك
- **بيانات ديناميكية**: تحديث فوري للمعلومات

### 💱 منصة التداول المتقدمة
- **تداول فوري**: شراء وبيع العملات والذهب مباشرة
- **أسواق متعددة**:
  - 🪙 البيتكوين (BTC/USD)
  - ⚡ الإيثيريوم (ETH/USD)
  - 🏆 الذهب (XAU/USD)
  - 💶 اليورو (EUR/USD)

- **معلومات تفصيلية**: أعلى وأقل سعر، حجم التداول
- **معنويات السوق**: مؤشرات صعودية وهابطة
- **واجهة بديهية**: سهلة الاستخدام للمبتدئين والمحترفين

### ✅ إدارة المهام المتقدمة
- **إضافة مهام بأولويات**: عالية 🔴 | متوسطة 🟡 | منخفضة 🟢
- **تصفية ذكية**: حسب الحالة والأولوية
- **إحصائيات حية**: معدل الإنجاز والمتبقي
- **حفظ تلقائي**: Local Storage للحفاظ على البيانات
- **حذف ذكي**: حذف المهام المكتملة بضغطة واحدة

### 🎨 واجهة مستخدم احترافية
- ✅ **تبديل الوضع**: Dark Mode و Light Mode
- ✅ **تصميم مستجيب**: يعمل على جميع الأجهزة
- ✅ **تجربة سلسة**: انتقالات جميلة وحركات احترافية
- ✅ **دعم العربية الكامل**: واجهة مصممة للغة العربية
- ✅ **أداء عالي**: تطبيق سريع وخفيف الوزن

## 🚀 البدء السريع

### المتطلبات
- Node.js 16+
- npm أو yarn

### الخطوات

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd youssef-trading-app
```

2. **تثبيت المكتبات**
```bash
npm install
```

3. **تشغيل السيرفر الحي**
```bash
npm run dev
```

4. **فتح المتصفح**
```
http://localhost:5173
```

5. **البناء للإنتاج** (اختياري)
```bash
npm run build
```

## 🎯 كيفية الاستخدام

### تبويب لوحة التحكم
1. اعرض ملخص حسابك وأرباحك
2. تابع جميع عملياتك الأخيرة
3. راجع الرسم البياني لتحليل الأسعار

### تبويب التداول
1. اختر العملة أو الأصل المراد التداول فيه
2. اختر نوع العملية (شراء أو بيع)
3. أدخل المبلغ المرغوب
4. اضغط تنفيذ العملية
5. راجع معنويات السوق للقرارات الذكية

### تبويب المهام
1. أضف مهمة جديدة مع تحديد الأولوية
2. صفّ المهام حسب الحالة أو الأولوية
3. ضع علامة على المهام المكتملة
4. احذف المهام المكتملة دفعة واحدة
5. راقب نسبة إنجازك

## 🎨 التخصيص

### تغيير الألوان
عدّل متغيرات CSS في `src/styles/App.css`:
```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --success-color: #10b981;
  --danger-color: #ef4444;
  /* ... المزيد من الألوان */
}
```

### إضافة أصول جديدة للتداول
عدّل مصفوفة `assets` في `src/components/TradingDashboard.jsx`:
```javascript
const assets = [
  { name: 'BTC/USD', price: '$64,230', change: '+2.4%', ... },
  // أضف أصول جديدة هنا
];
```
